# BndsCalcAux

!syntax description /AuxKernels/BndsCalcAux

The field value $b$ generated by this AuxKernel is the sum of the order
parameters $\eta_i$ squared.

\begin{equation}
b=\sum_i \eta_i^2
\end{equation}
